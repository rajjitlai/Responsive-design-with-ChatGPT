## Overview

This project showcases a responsive single-page website designed using code provided by ChatGPT along with custom enhancements. It features a mobile-friendly navbar, an interactive "Read More" section, and an auto-rotating image gallery.

## Features

- **Responsive Navbar** — Collapses into a hamburger menu on screens below 800px
- **Read More / Read Less Toggle** — Dynamic content visibility controlled via JavaScript
- **Image Carousel** — Auto-rotating image gallery with a 2-second interval
- **Mobile-First Design** — Fully responsive layout using CSS media queries

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/rajjitlai/Responsive-design-with-ChatGPT.git
   ```
2. **Open in browser**
   ```bash
   cd Responsive-design-with-ChatGPT
   open index.html
   ```
   Or simply open `index.html` directly in your preferred browser.
